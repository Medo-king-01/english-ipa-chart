## 🎯 نظرة عامة

**جدول IPA التفاعلي للإنجليزية** هو تطبيق ويب حديث ومتجاوب بالكامل، مصمم لمساعدة متعلمي اللغة الإنجليزية على إتقان **الأبجدية الصوتية الدولية (IPA)** من خلال:

- 🎧 **تشغيل الصوت** — الاستماع إلى النطق الصحيح لكل صوت
- 🖼️ **أدلة بصرية** — رؤية رسوم توضيحية لوضع الفم
- 🔍 **بحث وتصفية ذكية** — العثور على الأصوات فوراً
- 🎯 **وضع التمرين** — اختبار معرفتك باختبارات تفاعلية
- 🆚 **مقارنة الأصوات** — فهم الفروقات الدقيقة بين الأصوات المتشابهة
- 📝 **أنماط الإملاء** — تعلم كيفية كتابة كل صوت في الإنجليزية

جميع البيانات مخزنة محلياً في ملفات JSON، مما يجعل التطبيق خفيفاً وسريعاً وسهل التخصيص.

## 🚀 التثبيت والإعداد

### 1. استنساخ المستودع

```bash
git clone https://github.com/yourusername/english-ipa-chart.git
cd english-ipa-chart
```

### 2. التشغيل بخادم محلي

**الخيار أ — VS Code Live Server (موصى به)**

1. ثبت إضافة [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. انقر بزر الماوس الأيمن على `index.html` → **افتح بـ Live Server**

**الخيار ب — Python**

```bash
python3 -m http.server 3000
# زر http://localhost:3000
```

**الخيار ج — Node.js**

```bash
npx serve -p 3000
# أو
npx http-server -p 3000
```

**الخيار د — PHP**

```bash
php -S localhost:3000
```

### 3. التحقق من ملفات الصوت

جميع ملفات الصوت مُشار إليها بمسارات نسبية (`assets/audio/...`). تأكد من أن خادمك يخدم دليل `assets/` بشكل صحيح.

> ⚠️ **هام:** فتح ملفات HTML مباشرة عبر بروتوكول `file://` **لن يعمل** بسبب قيود CORS على تحميل ملفات JSON. استخدم دائماً خادماً محلياً.

## 📊 ملفات البيانات

### `ipa-data.json`

قائمة كاملة لأصوات IPA مع:

```json
{
  "id": 1,
  "symbol": "i:",
  "audioFile": "ipa/sound_i.mp3",
  "guideImage": "ipa-guides/im1.webp",
  "words": [
    { "word": "see", "audioFile": "ipa-samples/see.mp3" },
    { "word": "agree", "audioFile": "ipa-samples/agree.mp3" }
  ]
}
```

### `ipa-words.json`

الكلمة المثال الأساسية لكل رمز (تُستخدم في البطاقة الرئيسية):

```json
{ "i:": "bean", "I": "tip", "u:": "moon", ... }
```

### `ipa-descriptions.json`

أدلة النطق بال**عربية**:

```json
{ "i:": "صوت علة أمامي مغلق طويل، اللسان مرتفع قرب الحنك الصلب...", ... }
```

### `compare-data.json`

شروحات مقارنة مخصصة (مفتاح `symbol1-symbol2`):

```json
{ "i:-I": "/i:/ طويل واللسان مرتفع ومتوتر... أما /ɪ/ فقصير...", ... }
```

### `alphabet-data.json`

بيانات الأبجدية الإنجليزية:

```json
[
  {
    "letter": "A",
    "audioFile": "alphabet/a.mp3",
    "letterPhonetic": "/eɪ/",
    "smallLetter": "a",
    "sampleWord": "Apple",
    "wordPhonetic": "/ˈæpəl/"
  }
]
```

### `SPELLINGS` (مضمن في `home-page.js`)

أنماط الإملاء لكل رمز IPA (تُستخدم في النافذة التفصيلية):

```javascript
const SPELLINGS = {
  "i:": [
    { pattern: "ee", examples: "see, green, meet" },
    { pattern: "ea", examples: "eat, teacher, team" },
    // ...
  ]
}
```

## 🧠 البنية البرمجية لـ JavaScript

يتبع الكود **أنماط تصميم معيارية** مع فصل واضح للمسؤوليات:

### 1. `config.js` — الإعدادات المركزية

- **مصدر الحقيقة الوحيد** لجميع المسارات، الفئات، والثوابت
- `Object.freeze()` يمنع التعديلات العرضية أثناء التشغيل
- دوال مساعدة: `resolveAudioPath()`، `resolveImagePath()`

```javascript
const CONFIG = Object.freeze({
  paths: {
    alphabetData: 'data/alphabet-data.json',
    ipaData: 'data/ipa-data.json',
    audioDir: 'assets/audio/',
    // ...
  },
  symbolCategories: {
    vowels: ['i:', 'I', 'u:', ...],
    consonants: ['f', 'v', 'θ', ...]
  }
});
```

### 2. `data-service.js` — التخزين المؤقت للبيانات

- **نمط Singleton** مع ذاكرة تخزين مؤقت `Map` في الذاكرة
- يمنع طلبات الشبكة المكررة عبر الصفحات
- `loadMany()` تحمل ملفات JSON متعددة بالتوازي
- معالجة أنيقة للأخطاء (تُرجع `null` بدلاً من الرمي)

```javascript
const DataService = {
  load(path) { /* جلب + تخزين مؤقت */ },
  loadMany(paths) { /* Promise.all بالتوازي */ },
  clearCache() { /* إفراغ التخزين المؤقت */ }
};
```

### 3. `audio-service.js` — تشغيل الصوت

- **آلية تقليل الارتداد** تمنع النقرات المتكررة السريعة
- **معالجة الأخطاء** لـ `AbortError`، `NotAllowedError`
- مرجع Singleton للصوت الحالي قيد التشغيل
- تنظيف تلقائي عند أحداث `ended` و `error`

```javascript
const AudioService = {
  play(path) { /* تشغيل الصوت مع تقليل الارتداد */ },
  stop() { /* إيقاف الصوت الحالي */ },
  isPlaying() { /* يُرجع قيمة منطقية */ }
};
```

### 4. `theme-service.js` — الوضع الداكن/الفاتح

- **تفضيل مستمر** عبر `localStorage`
- يُزامن تلقائياً جميع أزرار `.theme-toggle` في الصفحة
- يُحدّث سمة `data-theme` على عنصر `<html>`

### 5. `sound-card-factory.js` — منشئ DOM

- **نمط المصنع** — ينشئ عناصر البطاقات بدون تأثيرات جانبية
- ثلاثة أنواع بطاقات: `buildIPACard()`، `buildAlphabetCard()`، `buildCompareCard()`
- فصل **الهيكل** عن **السلوك** (الأحداث تُربط في مكان آخر)

### 6. وحدات تحكم الصفحات

- `home-page.js` — 500+ سطر مع البحث، التصفية، وضع التمرين، النافذة المنبثقة، تنقل لوحة المفاتيح
- `alphabet-page.js` — عرض البطاقات، ربط الصوت، البحث، تتبع الماوس
- `compare-page.js` — تعبئة القوائم المنسدلة، منطق المقارنة، ربط الصوت

### 7. `particles.js` — خلفية Canvas

- **عدد جسيمات متجاوب** — يتناسب مع حجم الشاشة (مراعاة الأداء)
- **تفاعل الماوس** — الجسيمات تنفر من المؤشر
- **ألوان واعية بالسمة** — تبديل تلقائي بين الجسيمات السوداء/البيضاء
- **تأثير توهج ناعم** — تدرجات دائرية مع خطوط الاتصال

## 🔧 دليل التخصيص

### إضافة صوت IPA جديد

1. عدّل `data/ipa-data.json` وأضف كائناً جديداً:
```json
{
  "id": 45,
  "symbol": "new",
  "audioFile": "ipa/sound_new.mp3",
  "guideImage": "ipa-guides/im45.webp",
  "words": [
    { "word": "example", "audioFile": "ipa-samples/example.mp3" }
  ]
}
```

2. أضف الرمز إلى `CONFIG.symbolCategories` في `config.js`

3. أضف إدخالاً في `data/ipa-words.json`: `"new": "example"`

4. أضف إدخالاً في `data/ipa-descriptions.json`: `"new": "وصف بالعربية..."`

5. أضف أنماط الإملاء في كائن `SPELLINGS` في `home-page.js`

6. أضف شروحات المقارنة في `data/compare-data.json`

### تغيير الألوان

عدّل خصائص CSS المخصصة في `css/style.css`:
```css
:root {
  --color-accent: #d81b3f;   /* غيّر إلى لون علامتك التجارية */
  --color-gold: #c98b1a;     /* لون حروف العلة */
  --color-cyan: #0d7a9e;     /* لون السواكن */
}
```

### إضافة صفحة جديدة

1. أنشئ ملف HTML جديد (مثلاً: `new-page.html`)
2. انسخ هيكل الرأس/التذييل من الصفحات الموجودة
3. أنشئ ملف JS جديد في `js/` (مثلاً: `new-page.js`)
4. استورده في HTML بـ `<script src="js/new-page.js"></script>`
5. أضف الصفحة إلى التنقل في جميع الصفحات
